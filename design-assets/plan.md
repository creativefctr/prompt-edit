Recommended hero structure
1. Top announcement bar

Use this to create urgency and relevance.

Example:

Existing Student Offer: Get $110 in AI credits for just $98 — no subscription required.

Or:

Access top AI tools in one place. Pay only for what you use.

Keep it short. This should not compete with the headline.

2. Main hero headline

The headline should focus on the pain: too many tools, too many subscriptions.

Best direction:

Stop Paying for 10 Different AI Tools.

Access the Best AI Creation Tools in One Simple Marketplace.

Alternative headline options:

Create AI Images, Videos, Voiceovers, Ads, Music & More — Without Stacking Monthly Subscriptions.

The Pay-As-You-Go AI Marketplace Built for Content Creators.

My favorite is the first one because it clearly attacks the core problem and makes the offer easy to understand.

3. Subheadline

The subheadline should explain the model in plain language.

Recommended copy:

PromptEdit gives you one place to access leading AI image, video, audio, and editing tools. Buy credits, use the tools you need, and only pay for what you actually create.

This matches the page’s “grocery store for AI” positioning and credit-based model.

4. Primary CTA

Use action language tied to the offer.

CTA button options:

Get Started with AI Credits
Claim My Discounted Credits
Start Creating with PromptEdit
Try PromptEdit Today

For this specific page, I would use:

Claim My Discounted Credits

Under the button, add microcopy:

No subscription required. Use credits across any AI tool.

That line is important because “no subscription required” is one of the strongest selling points.

This is more emotionally exciting, but it should still include dashboard UI so people understand the business model.
